# Shakō

**Shakō** is a sleek, modern link-in-bio page built with Nuxt and Vue. It’s fully self-hostable and easily customizable to fit your personal brand.

## Key Features

- **Customizable:** Effortlessly adjust the appearance and functionality to match your style.
- **Self-hostable:** Deploy on your own server for complete ownership and privacy.
- **Modern design:** Clean, professional, and visually striking to make your links stand out.
- **Responsive:** Looks great on all devices, ensuring your links are accessible anywhere.
